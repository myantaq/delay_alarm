市販のドア用警報ブザーを改造し
警報が鳴り始めるまでの時間を設定する

材料
	ドア用警報ブザー
	マイコン AVR tiny13A
	高輝度LED
	配線

単4マンガン電池(450mAh)での電池寿命１年以上を目標とする
	基本的にマイコンはパワーダウンモードで動作させる
	リードスイッチの電源供給も不要時はカットする
	省エネと視認性向上のため高輝度LEDを使用する
	(マイコンのプルアップ出力インピーダンス約30kΩでドライブ)

	3V電源時
		ドア閉鎖LED消灯時 6uA
		ドア開放LED点灯時 40uA
		平均(多めに見積もって) 20uA

	単4マンガン電池 450mAh
		450mAh / 20uA = 22500h = 937days
		警報を鳴らさなければ電池寿命２年以上
